Reflection & Learning Journey

Throughout this Mobile Programming course, I have gained valuable experience in Android application development using Kotlin and Jetpack Compose. Starting from Lab 1, I learned how to design user interfaces and gradually explored user interactions, Material Design principles, navigation, and local data management. Each lab helped strengthen my understanding of modern Android development practices.

When completing Project 1 and Project 2, I combined the knowledge from previous labs into a complete healthcare application. I implemented multiple screens with smooth navigation, integrated network communication using Retrofit, utilized Android sensors to improve user experience, and connected Firebase for cloud data synchronization. These practical tasks enhanced my problem-solving skills and confidence in developing real-world mobile applications.

Overall, this learning journey has significantly improved my programming ability and understanding of mobile software engineering. It has also inspired me to continue learning advanced Android technologies and build more intelligent, user-friendly applications in the future.
